Planning & Design

Design objectives

BR1, I will be using a microbit as my embedded system. I will be using a DHT11 sensor as my analogue input and i will use the button A as my digital input. My system will have an alarm and the data collected by the sensor as its outputs. The microbit will start by pressing button A and after that it will run automatically .

BR2, the DHT11 sensor will be collecting temperature and moisture data which will then be saved to a SCV file and used in my risk of fungi spreading model

BR3, the microbit will be taking in data in real time and when the moisture levels reach over the threshold, the microbit will set off a warning sign. i will show in real time how it responds to things like windy conditions and very moist conditions.

AR1, my python model will calculate how high the risk of fungi spreading is according to the temperature and moisture. I will be using the data collected from the microbit . A graph will be created using matplotlib to show all the results in a visual manner and be easier to understand.

AR2, 1 what if the tempeture is high and the humidity is low, 2 what iff both temperature and humidity are high. I will be using outsourced data for these scenarios

AR3, I will have an alarm in my microbit when the humidity is high, displaying an image informing the user of the danger

Different project options considered

From completing the investigation stage, I have learned that the fungus that affects the ash tree needs elevated temperature and high moisture to thrive. So, for my microbit embedded system I knew I had to use a temperature sensor and a moisture sensor.

I found out that the micro bit has an imbedded temperature sensor inside but upon further research I found that it was a little bit faulty as the temperature would stay solid at around 20 degrees due to it being so close to the other components on the micro bit.

I was also investigating different moisture sensors and found that some would stop working after a short while

Due to this I started researching external sensors that I could connect to the microbit via a breakout board, I was able to find a sensor named DHT11 (a temperature and humidity sensor). This sensor was perfect for my project as it could measure both temperature and moisture.
